1. Copy boilerplate as new project folder

2. cd into project folder

3. npm init -y

4. Install dependencies
	ie: npm install express mongoose cors dotenv

5. git init ...to initialized

6. create .gitignore file
	add ignore paths: 
	.env 
	/node_modules

7. npx create-react-app APPNAME( standard is to use "client")

8. Rename Widget file names and content names where widget appears:
    widget.model.js
    widget.controller.js
    widget.routes.js
    server.js - (require....widget.route)
    test.http

9. Name your DB in .env file

TEST ROUTES

10. Start server
    nodemon server.js

11. test routes in test.http file


ADD CLIENT DEPENDENCIES

12. npm install bootstrap axios @reach/router


START CREATING FRONT END and COMPONENTS
